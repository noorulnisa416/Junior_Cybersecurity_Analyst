# Junior Cybersecurity Analyst — Career Path

A self-paced/Instructor-Led learning path from Cisco Networking Academy (NetAcad) designed to take you from cybersecurity fundamentals to entry-level, job-ready skills — culminating in a path exam and eligibility for the CCST Cybersecurity certification.

## 📋 Overview

| | |
|---|---|
| **Path Type** | Career Path (Course + Exam) |
| **Level** | Beginner → Entry-level |
| **Provider** | Cisco Networking Academy (NetAcad) |
| **Outcome** | Digital badge + eligibility for CCST Cybersecurity certification |

## 🗺️ Learning Roadmap

Follow the courses in order — each one builds on the skills from the last.

### 1. [Introduction to Cybersecurity](https://www.netacad.com/courses/introduction-to-cybersecurity?courseLang=en-US)
Explore the cybersecurity field and understand why it's a future-proof career choice.
- **Languages:** English, Arabic, Bahasa Indonesia, +18 more

### 2. [Networking Basics](https://www.netacad.com/courses/networking-basics?courseLang=en-US)
Learn the fundamentals of computer networking and how networks operate.
- **Languages:** English, Arabic, Chinese (Simplified), +9 more

### 3. [Networking Devices and Initial Configuration](https://www.netacad.com/courses/networking-devices-and-initial-configuration?courseLang=en-US)
Build on networking essentials with hands-on device configuration skills.
- **Languages:** English, Arabic, Chinese (Simplified), +6 more

### 4. [Endpoint Security](https://www.netacad.com/courses/endpoint-security?courseLang=en-US)
Learn to secure your network all the way to the edge.
- **Languages:** English, Arabic, Bahasa Indonesia, +12 more

### 5. [Network Defense](https://www.netacad.com/courses/network-defense?courseLang=en-US)
Learn to monitor and protect networks and evaluate security alerts.
- **Languages:** English, Arabic, Bahasa Indonesia, +9 more

### 6. [Cyber Threat Management](https://www.netacad.com/courses/cyber-threat-management?courseLang=en-US)
Learn cybersecurity governance and build skills to manage threats.
- **Languages:** English, Arabic, Chinese (Simplified), +8 more

### 🎓 Final Assessment: [Junior Cybersecurity Analyst Career Path Exam](https://www.netacad.com/exams/cybersecurity-pathway?courseLang=en-US)
Test your knowledge across the entire path and earn a digital badge verifying your skills.
- **Languages:** English, Arabic, +4 more

## 🏆 Next Steps After Completion

### [Cisco Certified Support Technician (CCST) Cybersecurity](https://www.netacad.com/resources/program-info/ccst-cybersecurity?courseLang=en-US)
Once you finish the career path, you're ready to pursue your entry-level cybersecurity certification.
